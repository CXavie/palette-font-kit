üöÄ Key Functionality
Hue-Locked Color Engine: Prevents the common "RGB jump" where lowering saturation to monochromatic levels resets the color picker to Red. The hue is preserved mathematically.

Mathematical Harmonies: Switch between different color relationship models:

Analogous: Colors adjacent on the wheel for a serene, comfortable design.

Monochrome: Varying tones and tints of a single hue for a clean, unified look.

Complementary: High-contrast pairings from opposite sides of the wheel.

Triadic: Three colors evenly spaced for vibrant, balanced diversity.

Accessibility First: Features a real-time WCAG Contrast Ratio calculator that checks the Primary color against the background and provides a visual "Pass/Fail" badge.

Typography Playground: Includes a curated selection of Google Fonts (Modern Sans, Elegant Serif, and Monospace) to test how colors interact with different typefaces.

Style Persistence: A built-in "Favorites" sidebar using localStorage so you can save and revisit palettes without needing a database.

Dark Mode Aware: Fully responsive UI that supports a dedicated Dark Mode toggle to see how your colors perform on deep backgrounds.
